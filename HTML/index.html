Difference between 'inline' and 'block-level' elements

Inline elements Block-level elements
An inline element only takes up as much width as necessary and does not start on a new line. A block-level element always starts on a new line and takes up the full width available.
Examples of inline elements include <span>, <a>, <strong>, <em>, <img>, and <input>. Examples of block-level elements include <div>, <p>, <h1> through <h6>, <ul>, <ol>, and <li>.
